Стиль появления букв A→W→S→M

Выбери анимацию для лоадера. Нажми на карточку для выбора.

A W S M

A — Scale + Fade

Каждая буква «вырастает» из точки. Чисто, минималистично.

Выход: шторка вверх (как page transition)
A W S M

B — Slide Up + цвета бренда

Буквы вылетают снизу, каждая — своим цветом (indigo → teal → coral). Энергично.

Выход: шторка вверх (как page transition)
A W S M

C — Typewriter + курсор

Буквы «печатаются» с мигающим курсором. Отсылка к code/terminal.

Выход: fade out

Анимации повторяются каждые 3 сек. Можно перезагрузить страницу чтобы сбросить.
