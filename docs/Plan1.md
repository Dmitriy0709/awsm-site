### 1. Текущий сайт AWSM (`/AWSM new/awsm-site/`)

#### Архитектура
- **4 отдельные страницы**: `/` (главная), `/cases`, `/services`, `/audit`
- Навигация ведёт на отдельные URL-адреса: каждый раздел — своя страница
- Роутинг: Next.js App Router, файлы в `app/cases/page.tsx`, `app/services/page.tsx`, `app/audit/page.tsx`

#### Цветовая система (источник: `globals.css`, `tailwind.config.ts`)
```
Base:           #FAFAFE  — основной фон
Surface:        #FFFFFF  — фон карточек
Surface-mid:    #F4F4FA  — чередующийся фон
Border:         #E4E2F0  — рамки
Primary:        #5A50DF  — electric indigo (ссылки, иконки)
Secondary:      #0EA888  — teal green (метрики)
CTA:            #D06830  — amber coral (кнопки)
Text Primary:   #17152E  — тёмно-синий
Text Secondary: #4A4870
Text Muted:     #9898B8
```

#### Шрифты (фактически в коде)
- Заголовки: `var(--font-display)` → Space Grotesk Bold
- Тело: `var(--font-body)` → Inter
- PRD предусматривал Syne + Plus Jakarta Sans — в итоге упрощено до Space Grotesk + Inter

#### Раздел главной страницы (файл `app/page.tsx`)
Порядок секций:
1. `HeroSection` — H1 + badge + stats-карточки
2. `SocialProofBar` — полоса цифр (27 500+ / 100+ / 12 000% / ТОП-1)
3. `VideoSection` — плашка видео
4. `PricingPreviewSection` — 3 карточки тарифов
5. `ProblemSolutionSection` — проблема/решение в 2 колонки
6. `AudienceSection` — 6 карточек «Кому подходит»
7. `HowWeWorkSection` — 4 шага
8. `CasesPreviewSection` — 3 кейса
9. `CtaSection` — финальная форма

#### Компоненты-секции по страницам
- `components/sections/home/` — 9 секций главной
- `components/sections/cases/` — CasesGrid, CasesHero
- `components/sections/services/` — FaqSection, PricingTable, ServicesHero
- `components/sections/audit/` — AuditForm, AuditHero

#### Визуальный стиль (текущий)
- **Техно / стартап**: glows, pulse-анимации, gradient text, dot-grid текстура
- Карточки: `card-glass` (border + shadow + hover lift)
- Кнопки: CTA коралл `#D06830` с box-shadow glow
- Анимации: Framer Motion на каждом элементе Hero
- Cursor: кастомный (CustomCursor.tsx)
- 3D-сцены: Three.js (запланированы)
- Метафора дизайна: «Digital Cartography» — ночной город, электрические потоки

#### Статус реализации
- ✅ NavigationBar (sticky, blur, hamburger)
- ✅ HeroSection (базовая без 3D)
- ✅ Дизайн-система (CSS-переменные, tailwind.config)

### 2. Сайт-вдохновение: parametr.space

#### Архитектура
- **Одностраничный**: всё содержимое — на одной странице, якорная навигация
- Меню ведёт к секциям через hash-ссылки, не к отдельным URL

#### Цветовая система
- **Светлый сайт**: белый/светло-серый фон (#FFFFFF / #F5F5F5)
- Текст: почти чёрный (near-black, ~#1A1A1A или #111111)
- Акцент: тёмный (нет ярких неонов, нет градиентов)
- Нет glows, neon, electric-эффектов

#### Типографика
- Крупные, жирные заголовки (700 weight) — приоритет читаемости над декором
- Большой контраст: заголовок жирный крупный vs тело лёгкое мелкое
- Modern sans-serif без засечек
- Числа-метрики: крупно, выразительно, без mono-шрифта

#### Навигация
- Минималистичная: текстовые ссылки на светлом фоне
- Нет фона-блюра, нет эффектов — просто чистые ссылки
- CTA кнопка в тёмном цвете (solid, без gradient)

#### Карточки и блоки
- **Flat design**: без теней, без glow, без border-radius > 12px
- Разделители: простые горизонтальные линии (1px)
- Отступы: щедрые (60–100px между секциями), создают «воздух»
- Нет стеклянных / glassmorphism эффектов

#### Кнопки CTA
- Solid тёмный (near-black) с белым текстом
- Без градиентов, без box-shadow
- Возможно мягкое скругление (8–12px)

#### Секции (одностраничная структура)
1. **Hero** — крупный заголовок «Девелопер для бизнеса» + KPI (2.5 млн м², 583 Га, 9 проектов)
2. **Форматы** — 4 карточки с иконками
3. **Проекты** — горизонтальный слайдер (6 проектов)
4. **Параметры эффективности** — слайдер с 4 преимуществами
5. **Решения** — аккордеон (малый / средний / крупный бизнес)
6. **Новости** — карточки с датами
7. **Подписка / Контакты** — форма + соцсети
8. **Футер**

#### Ключевые принципы aesthetics parametr.space
- **Бизнес-язык**: цифры говорят сами за себя, минимум украшений
- **Воздух и пространство**: секции не «налезают» друг на друга
- **Типографика как дизайн**: крупные числа и фразы — главные элементы
- **Строгость без холодности**: профессионально, но не корпоративно-скучно
- **Прогрессивные числа**: статистика видна сразу, до прокрутки
